## Add the Jitpack repository

<Tabs>
<Tab title="Past">

```groovy #4 build.gradle (root)
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
</Tab>

<Tab title="Gradle 8.0 or higher">

```groovy #4,11 settings.gradle
pluginManagement {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
</Tab>

</Tabs>
